HTML:

The full form of the html is hyper text markup language. It create the basic structure of web page. Tags are used to create the structure. The basic tag are head, html, body etc

CSS:

The full form of the css is Cascading Style Sheets. It use design the web page to look more attractive. It use by selecting html elements form html files. Types of css: inline , internal(style tag) and external(link tag).

JavaScript:

JavaScript use to make web pages responsive. It add logic inside the website. It apply by using selectors. script tag use to implement it inside html
